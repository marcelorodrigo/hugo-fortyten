The FortyTen theme is designed for simplicity, extensibility, and Hugo best practices.

**Key Features:**
- Responsive layouts with Tailwind CSS
- Custom navigation menus
- Modular Hugo template partials (edit header, footer, etc.)
- Localization-ready
- Asset pipeline for images, fonts, JS
- Clean, readable typography
